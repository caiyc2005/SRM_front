# 接口清单

> 所有接口统一使用 **POST** 方法（后端约定），响应格式统一如下：
> ```json
> { "success": true, "message": "操作成功", "data": [...] }
> ```

---

## 一、认证模块

### POST /api/login

**请求体：**
```json
{
  "userCode": "admin",
  "password": "password"
}
```

**响应体：**
```json
{
  "success": true,
  "token": "xxx",
  "user": {
    "userName": "admin",
    "userCode": "admin"
  }
}
```

## 二、用户管理（/api/User）

### POST GetUsers
获取用户列表

**响应体：**
```json
{
  "success": true,
  "data": [
    {
      "userID": "USR001",
      "userCode": "admin",
      "userName": "admin",
      "isDel": false,
      "memo": "系统管理员"
    }
  ]
}
```

### POST AddUser
新增用户

**请求体：**
```json
{
  "userCode": "zhangsan",
  "userName": "张三",
  "password": "123456",
  "memo": "备注信息"
}
```

**响应体：** `{ "success": true, "message": "用户添加成功" }`

### POST UpdateUser
修改用户

**请求体：**
```json
{
  "userID": "USR001",
  "userCode": "admin",
  "userName": "管理员",
  "password": "newpassword",
  "memo": "修改备注"
}
```
> password 可选，留空则不修改密码

**响应体：** `{ "success": true, "message": "用户修改成功" }`

### POST DeleteUser
禁用用户（软删除），不可逆

**查询参数：** `?id=USR001`

**响应体：** `{ "success": true, "message": "用户已禁用" }`

### POST GetRoles
获取角色列表（用于分配角色弹窗）

**响应体：**
```json
{
  "success": true,
  "data": [
    {
      "roleID": "R001",
      "roleName": "仓管员",
      "isDel": false,
      "memo": "仓库日常管理"
    }
  ]
}
```

### POST GetUserRoles
获取用户-角色关联列表

**响应体：**
```json
{
  "success": true,
  "data": [
    {
      "userRoleID": "UR001",
      "userID": "USR001",
      "roleID": "R001"
    }
  ]
}
```

### POST AddUserToRole
添加用户到角色

**请求体：**
```json
{
  "userID": "USR001",
  "roleID": "R001"
}
```

**响应体：** `{ "success": true, "message": "用户已添加到角色" }`

### POST RemoveUserFromRole
从角色移除用户

**请求体：**
```json
{
  "userID": "USR001",
  "roleID": "R001"
}
```

**响应体：** `{ "success": true, "message": "用户已从角色移除" }`

## 三、角色管理（/api/User）

> 与用户管理共用同个 Controller

### POST GetRoles
获取角色列表

**响应体：**
```json
{
  "success": true,
  "data": [
    {
      "roleID": "R001",
      "roleName": "仓管员",
      "isDel": false,
      "memo": "仓库日常管理"
    }
  ]
}
```

### POST AddRole
新增角色

**请求体：**
```json
{
  "roleName": "新角色",
  "memo": "角色描述"
}
```

**响应体：** `{ "success": true, "message": "角色添加成功" }`

### POST UpdateRoleStatus
启用/禁用角色

**请求体：**
```json
{
  "roleId": "R001",
  "isDel": true
}
```
> isDel = true → 禁用，isDel = false → 启用

**响应体：** `{ "success": true, "message": "操作成功" }`

### POST DeleteRole
删除角色（软删除）。有关联用户时拒绝删除

**查询参数：** `?id=R001`

**响应体（成功）：** `{ "success": true, "message": "角色已删除" }`

**响应体（失败）：**
```json
{
  "success": false,
  "message": "该角色下存在 N 个用户，请先移除所有用户后再删除"
}
```

## 五、采购订单管理（/api/Orders）

### POST CreateOrder
创建采购订单

**请求体：**
```json
{
  "SupplierID": "S001",
  "SupplierName": "宏盛钢材有限公司",
  "Materials": [
    {
      "MaterialID": "M001",
      "Qty": 10,
      "UnitPrice": 25.00
    }
  ],
  "Memo": "订单备注"
}
```

**响应体：**
```json
{
  "success": true,
  "message": "采购订单创建成功",
  "data": {
    "orderID": "xxx",
    "orderCode": "PO20240622123456",
    "status": 0,
    "createTime": "2024-06-22T10:00:00"
  }
}
```

### POST GetOrders
获取采购订单列表（待确认列表 + 采购订单查询共用）

**查询参数：** `?orderCode=xxx&supplierID=xxx&status=0&pageNum=1&pageSize=10`

**响应体：**
```json
{
  "success": true,
  "data": [
    {
      "orderID": "1",
      "orderCode": "PO20240601000001",
      "supplierID": "S002",
      "supplierName": "杭州电子科技有限公司",
      "status": "0",
      "materialCount": 2,
      "totalAmount": "580.00",
      "deliveryDate": "2024-07-15",
      "createTime": "2024-06-01 10:00:00",
      "deliveryNo": "",
      "materials": [
        {
          "index": 1,
          "materialCode": "M1011",
          "materialName": "物料1",
          "spec": "标准规格",
          "qty": 20,
          "unitPrice": 15.50,
          "amount": 310.00
        }
      ]
    }
  ],
  "total": 50
}
```

### POST ConfirmOrder/{orderId}
确认订单（待确认 → 已确认）

**路径参数：** `orderId` 订单ID

**响应体：** `{ "success": true, "message": "订单确认成功" }`

### POST GenerateDelivery/{orderId}
生成送货单（已确认 → 待发货）

**路径参数：** `orderId` 订单ID

**响应体：**
```json
{
  "success": true,
  "message": "送货单生成成功",
  "data": { "deliveryNo": "DN202406001" }
}
```
